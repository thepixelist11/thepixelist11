## Project Details:
### [Bracket](https://github.com/thepixelist11/bracket)
<details>
A Scheme-like Lisp interpreter written from first principles in TypeScript.

Features include:
- Advanced reader and runtime macro support.
- Comprehensive developer tooling and REPL allowing for source-code reconstruction, automatic usage documentation generation, environment inspection, and more.
- Custom-built Lexer with fully featured unicode support, form-level reading, nested comments, numerical base support, positional metadata, and more.
- Hand-written recursive descent parser with macro expansion and a unified error handling model
- Fully modular and data-driven system using simple objects to define standard library builtins, REPL commands, and reader macros allowing for extensibility and maintainability.
- Persistent environments allowing multiple files (or REPLs) to be run in sequence with the same environment.
- Feature detection and file directives allowing for backwards compatibility and fine-grained feature support.

[Repo](https://github.com/thepixelist11/bracket)
</details>

### [Truth Table](https://github.com/thepixelist11/truth-table)
<details>
A truth table CLI tool and library for parsing, evaluating, simplifying, and displaying logical expressions in multiple formats.

Features include:
- Parsing logical expressions with standard operators including AND, OR, NOT, XOR, IMPLIES, and IFF.
- Simplifying expressions involving constants and repeated variables.
- Generating and printing truth tables in multiple formats.
- Intermediate expression evaluation for detailed tables.

[Repo](https://github.com/thepixelist11/truth-table)
</details>
